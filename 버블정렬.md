# 버블정렬

### 바로 옆에 있는 값을 비교하여 더 작은 값을 앞으로 보내는 정렬방식

but 가장 효율성이 떨어지는 알고리즘이다.

```javascript
// javascript로 구현해보기

const bubbleSort = function(list) {
    let i = 0,
        j = 0,
        temp = 0;

    for (i = 0; i < list.length; i += 1) {
        for (j = 0; j < list.length - i; j += 1) {
            if (list[j] > list[j + 1]) {
                temp = list[j];
                list[j] = list[j + 1];
                list[j + 1] = temp;
            }
        }
    }

    console.log(list);
}

bubbleSort([1, 10, 5, 8, 7, 6, 4, 3, 2, 9]);
```

### 버블정렬의 시간 복잡도는?

```
10까지의 숫자를 비교 연산할 때에 계산 횟수
10 + 9 + 8 + ... + 1

=> 10 * (10 + 1) / 2

이것을 N으로 일반화 하면

=> N * (N + 1) / 2
* N이 무한히 커지는 경우 자연수 더하기, 나누기 연산은 무시된다.

즉 N * N
이를 빅오표기법으로 표기하면 O(N * N)

버블정렬의 시간 복잡도는 O(N^2)이다. (비효율적)

시간복잡도는 선택정렬과 같지만 가장 작은 값 하나만을 교체하는 선택정렬과는 달리
버블정렬은 매번 바로 옆의 값과 교체하는 작업을 수행하기 때문에 시스템이 작업해야하는 양이 더 늘어나기 때문에 실제로는 훨씬 더 비효율적이다

```
